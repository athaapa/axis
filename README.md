Axis is a 2D graphing calculator that lets you visualize 2D equations in a 3D world.

**What this repo contains**

-   **Project:** A Next.js frontend and visualization utilities under `src/`.

**Requirements**

-   **Node:** Recommended `>= 18.x`.
-   **Package manager:** `pnpm` (this repo includes `pnpm-lock.yaml`).

**Quick Setup**

-   Install dependencies:

```bash
pnpm install
```

-   Run the development server:

```bash
pnpm dev
```

-   Build for production:

```bash
pnpm build
pnpm start
```
